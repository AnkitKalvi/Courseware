# Building Your First Blog

Hey there! This is a self-paced lesson where you will work on practically implementing some of the most exciting concepts you have been learning in Front-end Web Development.


## Instructions for the lesson

In this project-based lab, you will build a simple webpage using HTML & CSS. The theme of the project is “My First Blog”.


## Here is the list of mandatory concepts to be used, while building your project

Cascade, Specificity, Inheritance, Organizing your CSS, CSS selectors, CSS selectors, Cascade, Specificity, Inheritance, CSS selectors, Box model, Backgrounds, Borders, Writing modes, Overflow, CSS values & units, Sizing items in CSS
