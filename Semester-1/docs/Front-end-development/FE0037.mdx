# Pre-read for Teacher-Student Interaction

Hey there! TSI will be an interactive live season, where teachers and students will be discussing crucial technical interview questions and go deeper into the concepts learnt previously by using Inquiry-based learning. 


## Technical Interview Questions



1. What is CSS selector specificity and how does it work?
2. What does * { box-sizing: border-box; } do? What are its advantages?
3. What's the difference between inline and inline-block?
4. What is the difference between a block-level element and an inline element? Can you provide examples of each type of element?
5. Can you explain the difference between px, em and rem as they relate to font sizing?
