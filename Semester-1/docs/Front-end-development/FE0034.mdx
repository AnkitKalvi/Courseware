# Building Your First HTML Table

Hey there! This is a self-paced lesson where you will work on practically implementing some of the most exciting concepts you have been learning in Front-end Web Development.


## Instructions for the lesson

In this project-based lab, you will build a simple webpage using HTML & CSS. The theme of the project is “Comparision blog”.


## Here is the list of mandatory concepts to be used, while building your project

Adding table header, Adding table row, Allowing cells to span multiple rows and columns, Providing common styling to columns, Adding table data, Adding a caption to the table, Adding structure, Nesting Tables.
