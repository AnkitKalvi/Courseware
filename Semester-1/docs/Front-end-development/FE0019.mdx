## What is Box Model?

As per [MDN](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/The_box_model), “Everything in CSS has a box around it, and understanding these boxes is key to being able to create more complex layouts with CSS, or to align items with other items.”.
