## What is List Styling?

As per [MDN](https://developer.mozilla.org/en-US/docs/Learn/CSS/Styling_text/Styling_lists), “Lists behave like any other text for the most part, but there are some CSS properties specific to lists that we need to know about, as well as some best practices to consider.”
